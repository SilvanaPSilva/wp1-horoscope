Welcome to ASTRO ✨

ASTRO is a simple horoscope web application created for people who enjoy zodiac signs, astrology, and daily predictions.

Our website allows users to explore horoscope forecasts for all twelve zodiac signs, including daily, tomorrow, and monthly predictions.

The goal of this project is to provide an interactive and modern experience where users can easily discover horoscope insights in a clean and responsive interface.
